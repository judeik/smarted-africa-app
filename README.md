# SmartEd Africa App

📱 **SmartEd Africa – A mobile-first learning platform empowering African students with AI tutors, localized lessons, teacher microlearning, and offline learning for IDP camps.**  

---

### Features

- AI Tutors for WAEC preparation
- Localized lessons in Hausa and other African languages
- Teacher microlearning modules
- Offline-first for IDP camps
- Mobile-first design

---

### 🛠️ Tech Stack

- ⚡ Frontend: Vite + React + TypeScript

- 🎨 UI: TailwindCSS + Bootstrap

- 🔐 Auth: JWT-ready with mock demo login/signup

- 📦 State: Zustand (lightweight global store)

- ⚙️ Build Tools: ESLint + Prettier + Husky

---

### Hackathon Build (2025)

This repository contains the hackathon build of SmartEd Africa.

---

### Setup

```bash
npm install
npm run dev
```
